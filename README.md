# 🚍 Travel Mate – AI Transportation Chatbot

Travel Mate is a simple AI-powered chatbot developed in Python using the OpenAI API. It assists users by answering transportation-related queries and providing basic travel assistance through a conversational interface.

This project was developed as a learning project to explore conversational AI, API integration, and Python programming.

---

## 📌 Features

- 🚆 Answer transportation-related questions
- 🚌 Interactive command-line chatbot
- 💬 Maintains conversation history
- 🤖 Uses OpenAI's GPT model for generating responses
- 🔄 Continuous conversation until the user types `quit`

---

## 🛠️ Technologies Used

- Python
- OpenAI API
- GPT-3.5 Turbo
- Command Line Interface (CLI)

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Travel-Mate.git
```

### 2. Navigate to the project

```bash
cd Travel-Mate
```

### 3. Install the required package

```bash
pip install openai
```

### 4. Add your OpenAI API Key

Replace:

```python
openai.api_key = "api-key"
```

with your own API key.

### 5. Run the program

```bash
python chatbot.py
```
